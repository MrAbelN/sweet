Quickstart - Web

  1. Follow native quickstart
  2. install the helper cli: cargo install forky_cli
    • More details on the cli page
  3. Update your test to do some web stuff: 
    web_sys::window().unwrap()
		.document().unwrap()
		.body().unwrap()
		.set_inner_html("<h1>This is a heading</h1>");
expect(window).get("h1")?
	.to_contain_text("This is a heading")?;
  4. run forky sweet
    • requires wasm-bindgen-cli
